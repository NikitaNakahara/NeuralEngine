# Neural Engine
##### A futuristic game engine with neurointerface support

## Install from GitHub
Open terminal and type `git clone https:\\github.com\NikitaNakahara\NeuralEngine`

### For ___Linux___
Type a commands
```
cd NeuralEngine
mkdir build
cd build
cmake ../src
make
```
You can find executable file in folder **bin**

### For ___Windows___
Launch cmake from build directory and open VSC++ Solution

### If you have ___Next language___ version
Type a commands
```
cd NeuralEngine
mkdir build
cd build
next -c .. -o bin/NeuralEngine
```
If your OS is ___Windows___, compiler put **.exe** automatically
